# MuseWig Atelier 产品说明

## 1. 产品定位

MuseWig Atelier 是一个面向假发选购场景的品牌型网站，核心目标不是单纯展示商品，而是帮助用户完成选择：

- 支持多类假发统一展示：日常、Cosplay、婚礼、舞台、修复
- 提供 3D 试戴式可视化体验
- 根据头围、脸型、场景、预算生成推荐
- 补齐护理、FAQ、信任与转化内容

## 2. 站点结构

- 首页 Hero：品牌定位、价值点、最匹配推荐
- 产品库 Catalog：按分类/关键词/推荐度筛选
- 3D 试戴工作室 Atelier：实时调整头围、蓬松度、长度、刘海、肤色
- 智能匹配 Smart Match：给出推荐原因和尺码建议
- 护理与售后内容 Care：降低用户对“买后难打理”的顾虑
- FAQ：回答尺码、修复款、Cosplay 逻辑等问题

## 3. 已实现功能

- React + TypeScript + Vite 独立前端项目
- Three.js / React Three Fiber 3D 假头模展示
- 假发商品数据结构化建模
- 场景、脸型、色系、头围、预算驱动的推荐评分逻辑
- 商品筛选、排序、详情切换
- 前视佩戴效果卡片
- 移动端响应式布局

## 4. 视觉方向

- 主色：酒红紫 / 玫瑰裸粉 / 香槟金
- 风格：精致、女性向、美妆感、轻奢但不冷淡
- 适配假发、美妆、发饰、造型服务站点气质

## 5. 下一步可扩展

